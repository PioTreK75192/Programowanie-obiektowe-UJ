**Widoki (przeglądarka):**
- `http://localhost:8000/products` — lista produktów
- `http://localhost:8000/categories` — lista kategorii
- `http://localhost:8000/customers` — lista klientów
- `http://localhost:8000/admin` — panel administracyjny EasyAdmin

**API (JSON):**
- `GET http://localhost:8000/api/products` — lista produktów
- `POST http://localhost:8000/api/products` — dodaj produkt
- `DELETE http://localhost:8000/api/products/{id}` — usuń produkt
- `GET http://localhost:8000/api/categories` — lista kategorii
- `POST http://localhost:8000/api/categories` — dodaj kategorię
- `DELETE http://localhost:8000/api/categories/{id}` — usuń kategorię
- `GET http://localhost:8000/api/customers` — lista klientów
- `POST http://localhost:8000/api/customers` — dodaj klienta
- `DELETE http://localhost:8000/api/customers/{id}` — usuń klienta
